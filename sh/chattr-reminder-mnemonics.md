# [`chattr(1)`](https://manpages.debian.org/stable/chattr) reminder mnemonics

| attribute | meaning                         |
| :---------| :-------------------------------|
| `a`       | **a**ppend only                 |
| `A`       | no**A**time                     |
| `c`       | **c**ompress                    |
| `C`       | no **C**opy-on-write            |
| `d`       | no **d**um(1)                   |
| `D`       | **D**irsync                     |
| `e`       | **e**xtents block mapping       |
| `E`       | **E**ncrypted                   |
| `F`       | case-insensitive                |
| `i`       | **i**mmutable                   |
| `I`       | **I**ndexing currently          |
| `j`       | **j**ournaling writes           |
| `m`       | not compressed                  |
| `N`       | stored in i**N**ode             |
| `P`       | **P**roject directory           |
| `s`       | **s**ecure delete               |
| `S`       | **S**ynchronous writes          |
| `t`       | no **t**ail merging             |
| `T`       | **T**op dir for orlov allocator |
| `u`       | **u**ndelete-able               |
| `x`       | direct-access (da**x**)         |
| `V`       | fs-**V**erity                   |
