Go templates blank spaces trimming

Basic example

foo:
    {{ if .foo }}
        foo is true
    {{ else }}
        foo is false
    {{ end }}
bar: whatever

Colored blank spaces with interactive toggling

foo:\n
    {{- if .foo -}}\n
        foo is true\n
    {{- else -}}\n
        foo is false\n
    {{- end -}}\n
bar: whatever
